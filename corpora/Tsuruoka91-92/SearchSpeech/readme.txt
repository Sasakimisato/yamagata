
■■ 鶴岡方言音声向け検索・再生ツールについて

作成日： 2007.12.12
更新日： 2010.08.18
作成者： 有本泰子



■起動する前に

この検索・再生ツールの構成は下記のとおりです。
本ツールは初期設定として下記のフォルダ構成のもと動作します。

	-+- SearchSpeech --+- readme.txt （ユーザマニュアル）
	  |                |
	  |                +- run.bat （実行ファイル）
	  |                |
	  |                +- SearchSpeech.jar （再生ツールアーカイブ）
	  |                |
	  |                +- playList92K.dat （回答リスト）
	  |                |
	  |                +- newZokusei92.dat （属性リスト）
	  |
	  |
	  +- wav ----+- 91_001K ------ 91_201K_001.wav 〜 91_001K_325.wav
	             |
	             +-   ：    ------                 ：
	             |
	             +- 92_498K ------ 92_498K_101.wav 〜 92_498K_630.wav


また、この検索・再生ツールの起動にはJava6（jre1.6.0以上）が必要です。



■起動・終了方法

- 起動
run.batをダブルクリックしてください。

- 終了
右上の「×」を押してください。


■操作方法

- 音声ファイルへのパス設定
「音声ファイルへのパス設定」ボタンを押し、音声ファイルが入っているフォルダ
（wav/）を指定します。

- 検索
検索条件を指定し、「検索」ボタンを押します。

- 検索条件のクリア
「選択解除」ボタンを押します。

- 音声の再生
質問から回答までの再生をするには、検索結果から聞きたい音声を選択し、「再生」
ボタンを押します。回答のみを再生するには、検索結果から聞きたい音声を選択し、
「回答再生」ボタンを押します。

- 再生の停止
「再生停止」ボタンを押します。（連続再生もストップします。）

- 検索結果一覧からの削除
検索結果の一覧から不要な行を削除したい場合は、その行を選択して「削除」ボタン
を押します。（元のデータが削除される訳ではありません。）

- 検索結果の保存
検索結果を保存したい場合は、「保存」ボタンを押してファイル名を指定します。
（カンマ区切りのテキストファイルとして保存されます。）
